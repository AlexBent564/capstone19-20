Welcome to the TreeViz project page! TreeViz is a group formed to help Dr. Kiona Ogle and Dr. Michael Fell with developing a user-friendly platform for visualising tree growth! Using the data that our clients have acquired, we can create an interactive environment for the user to be able to understand the life cycle of a tree based on several environmental factors.
